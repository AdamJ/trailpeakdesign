# Trail Peak — Marketing Site UI Kit

A click-thru recreation of Adam Jolicoeur's portfolio / marketing site using Trail Peak tokens. Single-page React prototype at `index.html` with in-file JSX components (Navigation, Hero, CaseCard, OpenSourceCard, Footer, ContactForm).

Screens covered:
- Home (hero + text-with-cards sections + open source + footer)
- Work (case study list)
- Writing (blog index)
- About
- Contact (form)

Built from the Figma `/Page/Home` frame. Interactions are cosmetic — nav clicks swap screens, the contact form shows a "Thanks" confirmation.
